Task 2: Pseudocode a computer task: Managing Files/folders on your computer!
  1. Go through all the files that are outdated or unnecessary
  2. Free storage by trashing unnecessary files/folders
  3. Name files by following the naming conventions
  4. Create folders or designate a place for all of the files
  5. Organize and order files that are from most important to least
  6. Back up any files
  7. Save files and put them in the right place when you first create them for easy access
