Understand the Problem


Set up the board.
Game loop starts.
Think of a word that will be guessed out by the other player (2).
Obtain remaining guesses from player (2).
If player (2) guesses incorrect, player will fail.
It adds a body part to the board if the player guesses incorrect.
Game loop restart if player falls and get all body parts on the hangman.
If player (2) guesses correct, player will success.
Remaining Guess is added to the board if player guess correctly.
If player gets the word correctly, the game ends.

Pseudocode the High Level Approach

Set up the players.

Establish a word.

If player guesses right, then finished.
If failed, loop game plays again till success.
